This code sets up a webpage that displays a map using the Mapbox GL JS library. It fetches real-time bus location data from the MBTA API and dynamically updates markers on the map to represent the buses. The markers are customized with different colors and icons based on the direction of the buses. The code periodically refreshes the bus data and updates the markers every 15 seconds. It uses JavaScript functions to create, move, and remove markers on the map based on the received bus data.

This project is an OAuth2 full-stack restaurant application hosted at titomazzetta.com. The app is completely SSL-secured and hosted in the cloud via a Digital Ocean Droplet. It leverages Google and GitHub OAuth2 for secure login, providing robust authentication and ensuring data privacy. Stripe is integrated for seamless payment processing, allowing users to securely make purchases. The app utilizes GraphQL queries for filtering food items and handling order data, with SQLite for lightweight and reliable data management. The back-end is powered by Node.js and Express, while the front-end is built using React to create a responsive, user-friendly interface. The entire application is containerized with Docker, allowing for easy deployment and scaling.
